Em muitos casos queremos desenhar muitas funções dentro de um mesmo grafico, isso pode ser facilmente feito com o uso de virgulas

{{ html }}

    
gnuplot> plot ~função~, ~função2~
por exemplo:
{{ html }}

    
gnuplot> plot sin(x),cos(x)

Resultado:


Claro podemos fazer o mesmo para splot

{{ html }}

    
gnuplot> splot sin(x),cos(x)

Resultado:


o problema começa quando ou as funções são muito complexas, ou temos que ter alguma edição especial em alguma função especifica, nesses casos plotar diretamente desse modo não resolve o problema, para isso temos o multiplot, que é uma ferramenta poderosa, porem tem seus problemas não podemos mecher no grafico interativamente usando mouse pois esse modo de multiplot "estampa a função"

Multiplot

para usar o multiplot devemos ir no gnuplot e aplicar

{{ html }}

    
gnuplot> set multiplot

multiplot>
logo que estiver nesse modo, todo e qualquer plot que for feito, vai ser "estampado" no grafico atual
{{ html }}

    
gnuplot> set multiplot

multiplot> splot x**2

multiplot> splot sin(x)
gera



se quiser "limpar" todos os plots ja feitos, pode ser usado o comando "clear" {% endblock %}
