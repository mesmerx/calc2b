Utilizaremos abordagens parecidas com plotagem em R² se não está familiarizado com ela olhe plotar uma função em R², não veremos como suavizar ou deixar esse grafico mais "bonito" isso está na sessão avançada"

Plotagem direta

Aqui com plotagem direta queremos dizer plotagem de uma função definita diretamente, mais a frente vamos aprender a gerar uma variavel e nessa variavel fazer a protagem

dentro do GNUPLOT rode

{{ html }}

    
gnuplot> splot ~função~
por exemplo:
{{ html }}

    
gnuplot> splot sin(x)
(obs: tem q ser em ingles se for usar função senoidal)

Resultado:


Claro como estamos em R³ podemos usar mais variaveis nesse grafico

por exemplo:
{{ html }}

    
gnuplot> splot sin(x*y)
(obs: tem q ser em ingles se for usar função senoidal)

Resultado:


esse grafico está claramente ruim, e deve ser melhorado mais a frente

Ajeitando o grafico (basico)

Os eixos X, Y e Z são ajeitados de maneira similar ao em R² usando xrange , yrange e zrange

{{ html }}

    
gnuplot> set xrange[min:max]

gnuplot> set yrange[min:max]

gnuplot> set zrange[min:max]
por exemplo:
{{ html }}

    
gnuplot> set xrange[2:5]

gnuplot> set yrange[2:5]

gnuplot> set zrange[-5:5]

gnuplot> gnuplot> splot sin(x*y)
gera



perceba que esse grafico está muito mais bonito, por mais que ainda deva ter mudanças

Definir funções

a definição de funções ocorre muito parecida com a em R², porem agora podemos utilizar sem definir explicitamente as variaveis x e y (obs: xy=/=x*y, xy não é entendido, pois xy é uma variavel nova, x*y é entendido como X vezes Y)

{{ html }}

    gnuplot> gnuplot> f(x,y) = sin(x*y) 
    gnuplot> gnuplot> splot f(x,y)


perceba que podemos utilizar funções como no caso de r²
{{ html }}

    gnuplot> gnuplot> f(x) = sin(x) 
    gnuplot> gnuplot> splot f(x)
